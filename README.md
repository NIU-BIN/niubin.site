# Calf Poster

<img style="margin: 20px 0" src="https://github.com/NIU-BIN/niubin.site/blob/main/cover.png"/>

如果你有掘金的文章，并且希望有一个自己的博客，那么这个项目就是为您准备的。

## 工具

框架：Next.js
Css框架：tailwindcss
markdown语法转换：markdown-it
动画库：framer-motion

## 功能

- [x] 首页贡献、最近文章
- [x] 文章搜索、分页、详情
- [ ] 项目展示

## 快速上手

1. 安装依赖

```bash
pnpm i
```

2. 开发启动

```bash
npm run dev
```

3. 构建打包

```bash
npm run build
```
